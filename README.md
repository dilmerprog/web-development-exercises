# 💻 Desafios de Programação

Bem-vindo ao meu repositório de desafios de programação!

Aqui reúno os exercícios, desafios e pequenos projetos desenvolvidos durante meus estudos, com o objetivo de praticar diferentes linguagens, tecnologias e conceitos de desenvolvimento de software.

---

## 🎯 Objetivo

Este repositório tem como finalidade:

- Praticar os conteúdos aprendidos em cursos.
- Resolver desafios de programação.
- Registrar minha evolução como desenvolvedor.
- Construir um portfólio de estudos.

---

## 📂 Estrutura

Cada pasta representa um desafio ou projeto desenvolvido durante os estudos.

Exemplo:

```text
📁 01_IA_Web
📁 02_Loja_Virtual
📁 03_Portfolio
📁 04_Sistema_Login
```

Dentro de cada pasta estão todos os arquivos necessários para o funcionamento do projeto.
